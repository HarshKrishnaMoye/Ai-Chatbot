## Features

- **Human-like Responses**: Receive responses that mimic human conversation, providing a realistic and engaging experience.
- **User Interaction**: Chat seamlessly with the chatbot, simulating a genuine conversation flow.
- **Minimalistic UI**: Enjoy a clean and straightforward user interface for chatting.

## Installation
1. Clone this repository to your local machine.
2. Navigate to the project directory.
3. Run `npm install` to install the dependencies.
4. Obtain your ChatGPT API key from OpenAI.
5. Set up your ChatGPT API key in the project configuration.
6. Run `npm start` to launch the application locally.
